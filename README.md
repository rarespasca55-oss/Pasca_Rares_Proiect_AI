# Analiza_Sentimentelor

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/rarespasca55-oss/Analiza_Sentimentelor/blob/main/Analiza_Sentimentelor.ipynb)

Acest proiect utilizează tehnici de **Natural Language Processing (NLP)** pentru a clasifica și analiza sentimentele extrase din postările de pe Twitter.

##  Funcționalități
* **Procesare Date**: Curățarea textului și pregătirea pentru modelare.
* **Analiză Statistică**: Vizualizarea distribuției sentimentelor.
* **Model de Predicție**: Clasificarea automată în sentimente pozitive, negative sau neutre.

##  Tehnologii Utilizate
* **Python** (Pandas, Scikit-learn, NLTK)
* **Google Colab** pentru mediul de dezvoltare.
* **SQLite** pentru stocarea și interogarea setului de date.

##  Structura Proiectului
* `Analiza_Sentimentelor.ipynb`: Notebook-ul principal cu tot codul.
* `Tweets.csv`: Setul de date brut.
* `database.sqlite`: Baza de date utilizată pentru analize.
* `requirements.txt`: Librăriile necesare pentru rulare.

## Cum se utilizează
Apasă pe butonul **"Open in Colab"** de mai sus. Odată deschis, rulează prima celulă pentru a descărca automat toate dependințele și fișierele necesare.
